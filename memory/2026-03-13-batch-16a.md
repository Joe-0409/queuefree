## Source Ports 设计

### CurrentUserSourcePort
**用途：** 获取当前认证用户 ID  
**使用方：** OrdersService / PaymentsService / QueueGuardService  
**当前 Adapter：** DemoCurrentUserSourceAdapter  
**未来 Adapter：** JWT/session-backed user resolver

### CatalogReadSourcePort
**用途：** 查询可订购的 SKU 信息  
**使用方：** OrdersService  
**当前 Adapter：** DemoCatalogReadSourceAdapter  
**未来 Adapter：** Prisma-backed products + product_skus reader

### AddressBookSourcePort
**用途：** 验证用户地址归属  
**使用方：** OrdersService  
**当前 Adapter：** DemoAddressBookSourceAdapter  
**未来 Adapter：** Prisma-backed user_addresses reader
