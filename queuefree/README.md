## 技术栈

- **前端**: React Native (Expo), React, Next.js
- **后端**: NestJS, Prisma, SQLite
- **状态管理**: Zustand, TanStack Query
- **UI**: 自定义组件库 + Storybook

## 快速开始

```bash
# 安装依赖
pnpm install

# 前端验证
pnpm verify:frontend-guardrails

# 启动移动端
pnpm dev:mobile

# 启动后端 API
cd services/api
pnpm prisma:generate
pnpm prisma:push
pnpm prisma:seed
pnpm build
node dist/src/main.js
```
